| ID     | Description                                              | Pré-conditions                                                                                                                                  | Étapes de test                                                                                                                        | Résultat Attendu                                                                   |
|--------|----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| TC_LOGIN_001 | Connexion réussie avec des identifiants valides (Happy Path) | L'utilisateur dispose d'un compte enregistré avec un email et un mot de passe valides. L'application est lancée et la page de connexion est affichée. | 1. Saisir un email valide dans le champ "Email". <br> 2. Saisir le mot de passe valide correspondant dans le champ "Mot de passe". <br> 3. Cliquer sur le bouton "Se connecter". | L'utilisateur est redirigé vers la page du tableau de bord.                        |
| TC_LOGIN_002 | Tentative de connexion avec un email invalide (Negative) | L'application est lancée et la page de connexion est affichée. L'utilisateur n'a pas de compte avec l'email saisi.                               | 1. Saisir un email invalide (qui n'existe pas dans la base de données) dans le champ "Email". <br> 2. Saisir n'importe quel mot de passe dans le champ "Mot de passe". <br> 3. Cliquer sur le bouton "Se connecter". | Le message d'erreur "Utilisateur inconnu" est affiché. L'utilisateur n'est pas redirigé. |
| TC_LOGIN_003 | Mot de passe vide - Bouton de connexion désactivé (Edge Case) | L'application est lancée et la page de connexion est affichée. L'utilisateur a saisi un email valide.                                         | 1. Saisir un email valide dans le champ "Email". <br> 2. Laisser le champ "Mot de passe" vide.                                         | Le bouton "Se connecter" est désactivé et ne peut pas être cliqué.               |
